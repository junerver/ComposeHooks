useAsync

@Composable
fun useAsync(block: SuspendAsyncFn): () -> Unit

This hook function accepts a suspend function as a parameter and returns a function to execute that suspend function. When you call the returned function, the suspend function will be executed within the coroutine scope of the current component.

// Declare a suspend lambda like a normal lambda
val async = useAsync {
    delay(1.seconds)
    setState(state + 1)
}
// Easy to use
async()

@Composable
fun useAsync(): AsyncRunFn

This is a hook function that simplifies the use of coroutine scope. Its usage is similar to the run function and is equivalent to scope.launch { }. This overload doesn't take any parameters and returns a function that can be used to execute suspend functions.

val asyncRun = useAsync()
asyncRun {
  // Do something asynchronously
  delay(1.seconds)
  setState { it + 1 }
}
